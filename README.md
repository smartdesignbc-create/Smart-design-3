---

## 🚀 Step-by-Step: GitHub → Vercel Deploy

### Step 1 — Create the folder on your computer
```
smart-design-salon/
  public/
    index.html       ← paste smart-design-salon-FINAL.html content here, name it index.html
  vercel.json        ← paste vercel.json content here
  README.md          ← paste README.md content here (optional but good practice)
```

### Step 2 — Push to GitHub
1. Go to github.com → New Repository
2. Name it: `smart-design-salon`
3. Set to **Public**
4. Click **Create repository**
5. Upload your files (drag and drop all 3 into the repo)
6. Click **Commit changes**

### Step 3 — Deploy on Vercel (free)
1. Go to vercel.com → Sign up / Log in with GitHub
2. Click **Add New Project**
3. Select your `smart-design-salon` repository
4. Leave ALL settings as default
5. Click **Deploy**
6. ✅ Done — your live URL will be: `https://smart-design-salon.vercel.app`

## 🔗 Share Demo Feature

- Set expiry: 1 day / 2 days / 7 days / 30 days
